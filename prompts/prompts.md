# 🤖 Prompts Utilizados na Análise de Dados

Este documento reúne os prompts utilizados para extrair insights da base de dados de vendas utilizando inteligência artificial.

---

## 📌 Prompt 1 — Análise Geral de Vendas

```text
Você é um analista de dados especialista em e-commerce.

Analise a base de dados de vendas fornecida e identifique:

1. Os produtos mais vendidos
2. Os produtos que geram maior receita
3. Produtos com baixa performance

Apresente os resultados de forma clara e objetiva, destacando padrões relevantes.
```

---

## 📌 Prompt 2 — Performance por Plataforma

```text
Você é um analista de dados.

Com base na base de vendas, analise o desempenho das plataformas:

- Shopee
- Etsy
- AliExpress

Identifique:

1. Qual plataforma gera mais vendas
2. Qual gera mais receita
3. Diferenças de comportamento entre elas

Apresente insights estratégicos.
```

---

## 📌 Prompt 3 — Análise por País

```text
Você é um especialista em análise de mercado global.

Analise os dados e identifique:

1. Países com maior volume de compras
2. Países com maior faturamento
3. Possíveis oportunidades de expansão

Apresente conclusões estratégicas para o negócio.
```

---

## 📌 Prompt 4 — Impacto dos Descontos

```text
Você é um analista de dados com foco em precificação.

Analise:

1. A relação entre desconto e volume de vendas
2. Se descontos maiores aumentam vendas
3. Possíveis impactos negativos na receita

Apresente recomendações sobre o uso de cupons.
```

---

## 📌 Prompt 5 — Perfil dos Clientes

```text
Você é um analista de comportamento do consumidor.

Com base na data de nascimento dos clientes:

1. Estime a idade média dos compradores
2. Identifique a faixa etária predominante
3. Relacione idade com comportamento de compra (se possível)

Apresente insights sobre o público-alvo.
```

---

## 📌 Prompt 6 — Análise Temporal

```text
Você é um analista de dados.

Analise o fator tempo nos dados:

1. Identifique os meses com maior volume de vendas
2. Detecte tendências de crescimento ou queda
3. Aponte possíveis sazonalidades

Apresente insights relevantes para planejamento estratégico.
```

---

## 📌 Prompt 7 — Prompt Completo (Análise Estratégica)

```text
Você é um analista de dados sênior especialista em e-commerce.

Analise a base de dados de vendas e forneça:

- Principais produtos e desempenho
- Análise por plataforma (Shopee, Etsy, AliExpress)
- Análise por país
- Impacto de descontos
- Perfil dos clientes
- Tendências ao longo do tempo

Além disso, apresente:

- Padrões identificados
- Problemas encontrados
- Oportunidades de crescimento
- Recomendações estratégicas para aumentar vendas

A resposta deve ser clara, objetiva e orientada para tomada de decisão.
```

---

## 🚀 Considerações

Os prompts foram estruturados para guiar a inteligência artificial na extração de insights relevantes, simulando análises realizadas por profissionais de dados em ambientes reais.

A abordagem foi dividida em análises específicas e uma análise geral consolidada, garantindo maior profundidade nos resultados.
